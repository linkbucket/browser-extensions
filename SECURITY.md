# Security Policy

## Reporting a Vulnerability

If you discover a security issue in the Linkbucket browser extension, please **do not** open a public GitHub issue.

Instead, report it privately by emailing **security@linkbucket.app**.

Please include:

- A description of the issue and its potential impact
- Steps to reproduce, or a proof of concept
- The extension version (see `manifest.json` or the browser's extensions page) and browser version
- Any relevant logs or screenshots

We aim to acknowledge reports within 72 hours and will keep you updated as we investigate and remediate. Once a fix is available, we'll coordinate disclosure with you.

## Supported Versions

Only the latest released version of the extension receives security updates.
